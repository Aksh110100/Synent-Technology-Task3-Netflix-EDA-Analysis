# Synent-Technology-Task3-Netflix-EDA-Analysis
## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Netflix Dataset using Python. The objective is to identify meaningful trends and patterns in Netflix content, analyze content distribution, understand release trends, evaluate content ratings, and generate insights through data visualization.

---

## Project Objectives

* Understand the structure of the Netflix dataset.
* Perform data cleaning and preprocessing.
* Generate summary statistics.
* Analyze content distribution across Movies and TV Shows.
* Identify trends in content releases over time.
* Examine content ratings and audience targeting.
* Create visualizations to support insights.
* Generate meaningful business and content-related recommendations.

---

## Dataset Information

The Netflix Dataset contains information about movies and TV shows available on Netflix, including:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre (Listed In)

The dataset consists of Netflix content records collected from the Netflix catalog and is widely used for exploratory data analysis and visualization projects.

## Key Findings

* Movies represent a significantly larger portion of Netflix content than TV Shows.
* Netflix content has grown rapidly in recent years.
* Most content belongs to modern release years.
* TV-MA and TV-14 are among the most common content ratings.
* Netflix offers content targeting a wide range of audiences and regions.
* Content diversity has increased over time, supporting global audience engagement.
